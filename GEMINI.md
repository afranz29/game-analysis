# WINDOWS SECURITY ANALYSIS

## Datasets
- pcapng: pcap captures from Wireshark.
- parquet: use parquet as the format for processed and enriched data.

## Tools Available
- osquery: `C:\Program Files\osquery\osqueryi.exe`.

## Environmental Assumptions
- Use PowerShell, not Bash. You are on Windows.
- Use uv for Python scripts, virtual environments, and package installation.

## Misc Guidances
- Don't delete any generated scripts. Save them all. 
- Look for existing markdown scripts in the directory to learn and update.
